.mad File Extension

What is a .mad file?

A .mad file is a Mobile Analytics Document created by MAD Designer, a business intelligence tool for iOS and macOS. These files store the complete definition of a mobile analytics report, including data source connections, query information, and the layout of the report itself.

More Information

The .mad file format is central to the MAD Designer application. It allows users to design, save, and share interactive reports. Each .mad file is a self-contained package that includes:

  • Data Connection Details: Information on how to connect to the underlying database, such as MySQL or Microsoft SQL Server.
  • Query Definitions: The SQL queries used to retrieve data for the report.
  • Report Layout: The visual design of the report, including the placement of charts, tables, labels, and images.
  • Formatting Rules: Custom formatting applied to data fields and other elements.

Because .mad files bundle all necessary components, they are highly portable. A user can create a report on their Mac and share the .mad file with a colleague, who can then open and view the interactive report on their iPhone or iPad.

How to open a .mad file

You can open .mad files with the MAD Designer application, which is available on the Apple App Store for iPhone, iPad, and Mac.
